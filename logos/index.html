Logo Guidelines

Logos

The Event Store logo represents the company, Event Store LLP. It’s made up of eight chevrons in the shape of an ouroboros.

If you wish to use the Event Store logo when talking about our company or products, please use only an official version from this page. Do not reshape, distort, crop or recolour the Event Store logo.

Four versions are available to suit different settings.

  • Colour: for use on plain, light backgrounds.
  • Colour inverted: for use on plain, dark backgrounds.
  • Blackout: for use on busy, light backgrounds.
  • Whiteout: for use on busy, dark backgrounds.

Ouro

Ouro is our mascot. He’s a baby ouroboros. He appears in many of Event Store’s publications and sites, almost always flying at a 30 degree counter-clockwise angle. Aren’t dragons the cutest creatures?

Please only use Ouro when talking about the Event Store. Show him in all his glory, or crop from the right and bottom edges. Ouro looks especially handsome set against a dark background (we like #222222).
